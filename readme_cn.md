   为什么要创建这个项目？ 百度paddlepaddle工程化不是太好，为了方便大家在各种端上进行ocr推理，我们将它转换为onnx格式，使用python/c++/java/swift 将它移植到各个平台。
   
   预计支持以下平台：
   
   Windows /Linux/ IOS/ android/Java
